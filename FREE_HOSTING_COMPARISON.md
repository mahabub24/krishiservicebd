## 🚀 Vercel + MongoDB Atlas Deploy

### Step 1: MongoDB Atlas (Free Database)
```bash
1. https://cloud.mongodb.com এ যান
2. Free cluster create করুন (M0 Sandbox)
3. Database user create করুন
4. Network access এ "Allow Access from Anywhere" (0.0.0.0/0)
5. Connection string copy করুন
```

### Step 2: Vercel Deploy
```bash
1. https://vercel.com এ যান
2. GitHub দিয়ে sign up করুন
3. "Import Project" ক্লিক করুন
4. GitHub repository import করুন
5. Framework: "Other" select করুন
```

### Step 3: Environment Variables
Vercel dashboard এ:
```
NODE_ENV=production
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/krishiservicebd
JWT_SECRET=your_secret_key
PORT=5000
```

### Step 4: Build Settings
```
Build Command: npm run build
Output Directory: client/build
Install Command: npm install && cd client && npm install
```

### Step 5: Domain Setup
```
1. Vercel "Domains" এ যান
2. krishiservicebd.com add করুন
3. DNS settings update করুন:
   - A Record: @ → 76.76.21.21
   - CNAME: www → cname.vercel-dns.com
```
